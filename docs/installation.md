# Installation

+++ pip
```bash
pip install capmonster_python
```
+++ pipx
```bash
pipx install capmonster_python
```
+++ poetry
```bash
poetry add capmonster_python
```
+++

!!!warning To use the legacy v3 syntax
```bash
pip install capmonster-python==3.2
```
!!!
