# 🧠 Capmonster Python

Welcome to the documentation for **Capmonster Python** — a modern, fully typed, async-ready SDK
for [Capmonster.Cloud](https://capmonster.cloud/).

This site will help you integrate with various CAPTCHA challenge types such as reCAPTCHA, Turnstile, GeeTest,
Image-to-Text, and more.

---

## 🚀 Quick Start

Jump into your first integration:

- [Installation](installation.md)
- [Examples](examples.md)
- [API Reference](reference/overview.md)

## 📦 Features

- ✅ Async-first, httpx-based HTTP client
- ✅ Full Pydantic v2 data validation
- ✅ Task abstraction for all Capmonster types
- ✅ Clear and typed API responses
- ✅ Retry logic and error wrapping
- ✅ Fully documented public API
